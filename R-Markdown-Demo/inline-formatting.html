Authoring A Book with R Markdown : Demo 1

2.1 Inline formatting

2.1.1 Italic

You can make text italic by surrounding it with underscores or asterisks, e.g., _text_ or *text* renders text.

2.1.2 Bold face

For bold text, use two underscores (__text__) or asterisks (**text**) which renders text.

2.1.3 위/아래 첨자

~로 둘러 쌓인 텍스트는 아래 첨자가 된다(예, H~2~SO~4~는 H2SO4). 마찬가지로, ^로 둘러 쌓인 텍스트는 윗 첨자가 된다(예, Fe^2+^는 Fe2+).

2.1.4 인라인 코드

인라인 코드를 텍스트로 표시하기 위해서는 두 개의 백틱을 `code`처럼 사용한다. `code` 은 code와 같이 출력된다.1

2.1.5 작은 대문자

작은 대문자(Small caps)는 HTML의 태그인 span을 사용하여 생성할 수 있다. 예를 들어, <span style="font-variant:small-caps;">Small Caps</span> 은 Small Caps와 같이 출력된다.

NOTICE!

그런데, Small Caps를 사용하려면 style.css 파일에 다음의 내용이 등록되어 있어야 함.

.smallcaps {
  font-variant: small-caps;
}

2.1.6 링크와 이미지

링크는 [text](link) 형태로 생성한다 .예를 들어, [RStudio](https://www.rstudio.com)는 RStudio에 대한 링크를 생성해 준다.

마찬가지로, 이미지에 대한 구문도 유사한데, 단지 맨 앞에 감탄부호(!)를 붙여 준다. 예를 들어, ![alt text or image title](images/knit-logo.png) 는 와 같이 이미지를 삽입해 준다.

2.1.7 각주

각주는 캐럿 기호(^) 다음의 대괄호([]) 안에 기입한다. 예를 들어, ^[This is a footnote.]는 각주2와 같이 출력된다. We will talk about citations in Section 2.8.

  1. 백틱을 단순히 기호로 포함시키고 싶다면, 백틱의 왼쪽에 두 개의 백틱을 더 추가해 주면 된다. 예를 들어 `` `code` `` 라고 입력하면 `code`가 출력된다.↩︎

  2. This is a footnote.↩︎
